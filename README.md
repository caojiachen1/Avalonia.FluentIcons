## FluentIcons

An Avalonia icon control library, repackaged from the gallery of [FluentIcons](https://github.com/davidxuang/FluentIcons), adapted for use as a standalone Avalonia library.

The icons are sourced from [Microsoft Fluent UI System Icons](https://github.com/microsoft/fluentui-system-icons).
